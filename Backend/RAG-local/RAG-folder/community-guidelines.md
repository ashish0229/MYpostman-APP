# MYpostmate Community Guidelines & Moderation Master Document

## 1. Core Platform Rules
* **No Toxicity or Harassment:** Any content scoring above 0.85 on the Hugging Face toxicity detector will be automatically flagged for review.
* **Spam Prevention:** Users cannot post more than 5 times within a 10-minute window. Promotional links must be tagged with #promo.

## 2. The Reputation Score System
* Every user starts with a reputation score of 100.
* **Penalties:**
  * First flagged toxic post: -10 points and a warning.
  * Second flagged post: -30 points and a 24-hour mute.
  * Score drops below 40: User loses posting privileges entirely.

## 3. Edge Cases for AI Moderation
* **Coding Discussions:** Using words like "kill the process", "terminate", or "execute" in the context of programming (Java, Node.js, Python, etc.) is perfectly safe and should NOT be flagged as violence.
